# Human Copy Landing V3 Premium

Standalone Vite + React + TailwindCSS landing page preview.

## Install

```bash
npm install
```

## Dev

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Deploy hint

You can deploy the `dist` folder to any static host (e.g. GitHub Pages, Netlify, or Vercel).
