# Human Copy Landing V3 Premium

Standalone Vite + React + TailwindCSS landing page preview.

## Install

```bash
npm install
```

## Dev

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Deploy

Pushes to `main` automatically build and deploy the site to GitHub Pages via Actions.
